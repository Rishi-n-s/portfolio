# 🌌 Interstellar Portfolio: Rishi Solanki /Maruti

A premium, high-performance personal portfolio featuring an immersive **3D Interstellar Workspace**, **Motion-Controlled Flight**, and a **Glassmorphism Design System**.

![Portfolio Preview](certifications.png)

## 🚀 Key Features

### 🛠️ Core Portfolio
- **Modern Glassmorphism**: A sleek, translucent UI built with vanilla CSS.
- **Dynamic Theme Engine**: Seamless switching between Deep Space (Dark) and Nebula Light modes.
- **Interactive 3D Cards**: Smooth tilt effects using `Vanilla-Tilt.js` for all project showcases.
- **Custom Cursor System**: A responsive, animated cursor that reacts to interactive elements.

### 🪐 Interstellar Workspace (Three.js)
- **Cinematic 3D Scene**: Navigate through a procedurally generated starfield and solar system.
- **Project Planets**: Each portfolio project is represented as a unique planet with custom textures and glows.
- **Spaceship Navigation**: Control a realistic 3D ship to explore the system.
- **Motion Controls (Gyroscope)**: On mobile, tilt your phone to steer the ship and tap to engage thrusters.

### 🏆 Achievement Gallery
- **Premium Gallery**: A dedicated space for verified certifications and badges.
- **Hover Reveal**: 3D "Pop-out" effects that reveal full certificates without clipping.
- **Immersive Lightbox**: View achievements in high resolution with a single click.

## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **3D Engine**: [Three.js](https://threejs.org/)
- **Interactivity**: [Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/)
- **Icons**: Font Awesome 6.4
- **Typography**: Google Fonts (Outfit & Playfair Display)

## 📱 Mobile Controls

When viewing the **3D Workspace** on a mobile device:
1. Tap **"Enable Motion Controls"**.
2. Grant permission for **Motion and Orientation** (requires HTTPS).
3. **Tilt phone** to steer and **Tap & Hold** to fly forward.
